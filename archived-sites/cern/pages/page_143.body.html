Status of "Erwise" browser

A multifont multiwindow browser. Editor features may be introduced later. Neat formatting.
Authors:
The OHT-Erwise (sic) team in Helsinki.
Status:
Extremely pre-documented (in Finish). Serious coding started around 1-March 1992. Alpha release available by anonymous FTP from info.cern.ch -- binaries only (sun4 works, decstation too, display requires motif) as of 15 April 1992. Source code released see www-talk August 92. No support
