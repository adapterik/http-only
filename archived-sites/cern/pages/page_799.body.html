FIND Gateway

These notes are about the C part of the gateway, and about running it on VM.

The C part of this is composed of the files HTDaemon.c and FINDGate.c (also HTTCP.c)

On VM

To build, use EXECs CC2 (to compile each module) and DLOAD (to link).

To capture log on VM (use # if the program is running):

#CP SPOOL CON START bernd

#CP DISC
