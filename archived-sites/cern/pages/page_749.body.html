X11 browsers for W3

Under X11, the following possibilities exist for accessing W3 data. See the list of W3 products.
  • The Viola project provides an interface to W3 (violawww)
  • The Erwise browser, running using Motif;
  • The tkWWW browser using the TK toolkit
  • The MidasWWW browser using Midas gadgets;
  • Dan Connolly at Convex, Inc. has developped a browser under motif using Tcl. unfortunately not public domian.
  • Dan has made available code to make Frame into a HTML editor.
Other interesting possibilities for this include building one on top of
  • The Andrew toolkit (ATK) from Carnegie Mellon University. (The sources are released with X11).
  • The TMlib library from someone at Apollo. The distribution terms are basically free. I have the code and documentation.
  • The "interviews" C++ toolkit on top of X11
  • The C++ wrappers for Motif which have been made sort of public (but would need licensing agreements) from univerity of Lowell.
  • The Tcl toolkit, as used by Dan. A Tcl hypertext widget is now available from somewhere.
