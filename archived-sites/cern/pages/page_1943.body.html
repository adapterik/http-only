Unix "path"

This is the set of directories which the system looks in for executable files. You can find out what your path is by the command echo $path 

  Normally the path is set by a command in your login shell script .
  Remember that if you change your path, then you must type
  			rehash
  
  

  so that the shell rebuilds the set of executable files which it will
  accept as commands.
