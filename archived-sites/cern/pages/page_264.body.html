Guide

The Guide product is marketed by OWL , though originally developed in the research environment (under unix) by Peter Brown .

The OWL version runs on Macs and PCs using WINDOWS/3 though a beta test version running on unix using X exists. Maybe we could test it.

IDEX is a product which uses Guide but includes (in the same program) a document manger with full text search capability. It runs on PCs, and uses existing remote file access mechanisms to allow a distributed system. The documentation manger creates UIDs, and will convert a UID into a path.

Transfer format

Guide uses an SGML DTD they call HML. This would be very interesting to compare with HyTime's efforts, and WWW's "HTML".
