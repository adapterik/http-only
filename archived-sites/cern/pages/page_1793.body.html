Global Variables under VM/CMS

(These are the equivalent of environment variables under unix.)

A particular GLOBALV table (CENV) is used to hold the C environment. The command to set a variable VVVV to a value XXXX is

GLOBALV SELECT CENV SET VVVV XXXX 

  for example
  		GLOBALV SELECT CENV SET WWW_HOME http://info.cern.ch/junk.html
  
  

  Beware of the fact that CMS puts everything into upper case.  It is
  not obvious how to get around this problem without writing a little
  REXX exec to do what you want.
