Finding HTML examples on the web

If you are looking for examples of HTML to copy, just browse the web for something which looks like the effect you want to achieve. Then get the source for that document, virtual or real (you may never know).

Most browsers have teh ability to give you the source text.

The line mode brower's SOURCE prefix on a > or | command for example can be used, or you can use a -source option on the command line.

Some of the GUI browsers have an equivalent facility.
